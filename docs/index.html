Instalacao

Baixe, instale e abra pelo Menu Iniciar

O instalador MSI inclui o aplicativo e dependencias principais. Na primeira abertura, o IMD cria um arquivo de configuracao inicial para voce ajustar pelo proprio painel.

  1. Baixe o MSI Use sempre a versao mais recente publicada no GitHub Releases.
  2. Instale no Windows O app fica no seu usuario e cria um atalho no Menu Iniciar.
  3. Configure suas pastas Abra o painel, ajuste a planilha, pasta de musicas e formato de audio.

Perguntas

Antes de instalar

O IMD roda na internet?

Nao. Ele abre uma pagina local em 127.0.0.1, dentro do seu computador.

Preciso instalar Python?

Nao. O instalador foi pensado para levar o app pronto para usar.

Posso mudar as configuracoes?

Sim. O painel tem configuracoes basicas e avancadas em uma tela propria.

Existe alguma responsabilidade de uso?

Use apenas com conteudos e direitos que voce pode acessar, baixar ou converter.
